# Berberos — Premium Barbershop Website (Static Landing Page)

Berberos is a modern, mobile-friendly **barbershop / grooming salon website**. It’s built as a **static website** (HTML + CSS + JavaScript) and designed to showcase a premium brand experience: services & pricing, a visual gallery, an about section, and clear contact / WhatsApp call-to-actions.

This repository can be used as a ready-to-deploy template for a barbershop (or any local service business) that wants a clean online presence and a fast path to:

- **Look trustworthy and premium** on first impression
- **Convert visitors into appointments** (WhatsApp, phone, address, opening hours)
- **Show services & pricing** clearly
- **Show the place and the work** through a gallery

---

## What the website includes

- **Sticky navigation** with smooth scrolling between sections
- **Hero slider** with multiple full-screen slides
- **Services section** (cards with descriptions and pricing)
- **Gallery section** (image grid)
- **About section** with stats (experience, customers, satisfaction)
- **Contact section** (address, phone, email, WhatsApp link, working hours)
- **WhatsApp floating button** for quick messaging

---

## Tech stack

- **HTML** (`index.html`)
- **CSS** (`style.css`) + responsive layout
- **JavaScript** (`script.js`) for:
  - hero slider
  - navbar scroll effect
  - mobile menu toggle
  - smooth scrolling
  - simple reveal animations
- External UI assets:
  - **Bootstrap 5** (CSS)
  - **Font Awesome** (icons)
  - **Google Fonts**

## How customers can get a website like this (and use it to their advantage)

If you’re a business owner and want a website like Berberos, the process is typically:

1. **Brand setup**
   - Your logo, colors, fonts, and tone of voice
   - Photos of your shop and your work (high quality photos make a big difference)

2. **Content & sections** (the parts that drive conversions)
   - Services + pricing
   - “About” story and trust signals (years in business, customer count, reviews)
   - Contact details, map link, and a clear “Book now” action

3. **Customization**
   - Replace images in `images/`
   - Update texts, services, and prices in `index.html`
   - Update colors and spacing in `style.css`
   - Update WhatsApp phone number / contact details

4. **Publishing**
   - Connect a domain (e.g. `yourbarber.com`)
   - Deploy to a static host (GitHub Pages, Netlify, Vercel, or any shared hosting)

### Benefits for a local business

- **More appointments**: a “WhatsApp / Call now” flow removes friction
- **Better first impression**: premium design builds trust
- **Always online**: customers can see services and prices 24/7
- **Stronger brand**: consistent visuals across Instagram, Google Maps, and the website

## Running locally

Because it’s a static site, you can open `index.html` directly in a browser.

For best results (and to avoid local browser restrictions), run a simple local server:

```bash
# Python 3
python -m http.server 8000
```

Then open:

- `http://localhost:8000`
